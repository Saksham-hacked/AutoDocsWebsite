Frequently asked questions

What counts as a doc PR?
A doc PR is any pull request AutoDocs opens to update your documentation. One commit can produce one PR updating multiple doc files — that counts as one doc PR.
Does AutoDocs work with private repositories?
Yes. AutoDocs installs as a GitHub App and works with both public and private repositories. Your code is never stored — only LLM-generated summaries are retained.
Which languages and frameworks are supported?
Any language. AutoDocs reads your source files and diffs — it doesn't rely on language-specific parsers or plugins. JavaScript, Python, Go, Rust, Java — all work out of the box.
Can I review docs before they merge?
Always. AutoDocs opens a pull request — you review the generated documentation and merge when you're happy. Low-confidence sections are flagged with a warning in the PR description.
Is there a free trial for Pro?
Yes — Pro includes a 14-day free trial, no credit card required. You'll be notified before the trial ends.
