Features

Built for real codebases.

🔍

Semantic Memory

Every file summary is embedded in pgvector. AutoDocs finds related files across your entire codebase, not just what changed.

⚖️

Confidence Scoring

Every generated section includes a High / Medium / Low confidence rating. Low confidence sections flag themselves for human review.

🎯

Smart Classification

Rule-based + LLM classification detects new routes, env vars, schema changes, and dependency updates — routing each to the right doc.

🔒

Source References

Every generated paragraph cites its source file and line numbers. You can always verify what the AI wrote and why.

⚡

Fast Exit Logic

Minor refactors and comment-only changes are detected and skipped instantly. No wasted LLM calls, no noise PRs.

🌐

Any Stack

Works with JavaScript, Python, Go, or anything else. AutoDocs reads your code — it doesn't need a plugin for your framework.
